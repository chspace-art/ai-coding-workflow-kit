# UI Verification

## Verification gate

Before asking for user review:

1. Start or identify the correct dev server.
2. Confirm the page identity, not just the port.
3. Run available build or type checks.
4. Open the page in a real browser.
5. Verify the page is not blank and the main interaction works.

Treat a successful build as necessary, not sufficient.
