# Code of Conduct

This project aims to be welcoming, practical, and respectful.

## Expected behavior

- Assume good intent.
- Share evidence and reasoning without contempt.
- Critique ideas and implementations, not people.
- Help newcomers understand the workflow instead of guarding it.

## Unacceptable behavior

- Harassment or personal attacks.
- Publishing private data, secrets, or unredacted user material.
- Dismissing questions in a hostile or mocking way.
- Using contribution channels for spam or unrelated promotion.

## Maintainer guideline

When a discussion gets heated, slow the thread down, restate the technical disagreement, and move back to specific evidence and reproducible behavior.
