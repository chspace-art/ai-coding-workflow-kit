# AGENTS Example

Use this file as a starting point for a repo-local workflow contract.

## Startup

- Read `docs/LESSONS_LEARNED.md`
- Read `docs/memory/COMMON.md`
- Read matching topic memory
- Read `.agents/skills/INDEX.md`
- Check recent summaries in `share/archive/`

## Task flow

- Start meaningful tasks with `workflow_runner.py start`
- Confirm requirement understanding before the first dev record
- Record `dev -> mentor -> qa`
- Complete the task with a normalized summary

## UI rule

- Open the real page in a browser before asking for review
- Do not treat HTTP 200 or a successful build as proof that the UI works
