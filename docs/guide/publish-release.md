# Publish the First GitHub Release

If the release tag already exists, publishing the release in the GitHub UI is very short.

## Tag

Use:

```text
v0.1.0-alpha
```

## Release title

Use:

```text
v0.1.0-alpha
```

## Release notes source

Copy from:

```text
RELEASE_NOTES_v0.1.0-alpha.md
```

## GitHub page

Open:

```text
https://github.com/chspace-art/ai-coding-workflow-kit/releases/new?tag=v0.1.0-alpha
```

## Recommended options

- mark it as a pre-release
- keep it published, not draft

After that, the repository will have a visible first release page in addition to the pushed tag.
